# SHOWCASE UI PIPELINE

Bu dosya vitrin modulunun storefront ve admin UI akislarini takip eder.

## Modul Ayrimi

- Kategori: taksonomi, SEO, urun siniflandirma
- Vitrin: landing kurgusu, arkaplan, etiketler, one cikan urunler ve secili urun havuzu

## Tamamlananlar

- [x] Admin sidebar'a `Vitrinler` modulu eklendi
- [x] Vitrin listesi UI omurgasi eklendi
- [x] Vitrin create/edit formu eklendi
- [x] Formda arkaplan, etiket, kategori havuzu, one cikan urunler ve home gorunurlugu alanlari acildi
- [x] Home'da 6 vitrinlik giris bandi eklendi
- [x] Vitrin detail storefront hero sahnesinin ilk versiyonu eklendi
- [x] Admin formunda ikon preview alani eklendi
- [x] Admin formunda 7 urunluk slot yapisi eklendi
- [x] Slot bazli popup urun secimi ve surukle-birak ilk versiyonu eklendi

## Acik Isler

### Storefront

- [x] Home vitrin modulu icin ayri section tasarlandi
- [ ] Home vitrin kartlarini arkaplan thumbnail/preview ile zenginlestir
- [x] Vitrin detay hero sahnesinin ilk tasarimi kuruldu
- [x] Solda filtre paneli, sagda vitrin sahnesi ve altta urun grid duzeni kuruldu
- [ ] Mobil duzende vitrin filtre davranisini netlestir

### Home

- [x] Home'daki mevcut kategori odakli giris yuzeyine vitrin modulu eklendi
- [ ] `ShowOnHome` aktif vitrinler icin slider, grid veya highlight duzeni sec
- [ ] Vitrin kartlarinda arkaplan, etiket ve CTA hiyerarsisini belirle

### Admin

- [x] Arkaplan seciciyi daha gorsel hale getir
- [x] One cikan urunler icin surukle-birak siralama ilk versiyonunu ekle
- [ ] Vitrin yayinda/yayinda degil durumunu liste satirinda daha gorunur yap
- [ ] Popup urun secicide kategori chipleri, secili urun etiketi ve remove aksiyonu ekle

## Bir Sonraki UI Sira

1. Home vitrin modulu section tasarimini ekle
2. Vitrin detay sayfasinin storefront iskeletini kur
3. Admin formunda arkaplan secimini gorsellestir
